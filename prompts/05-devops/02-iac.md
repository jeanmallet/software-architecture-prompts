# Prompt: Infraestrutura como Código (IaC)

## Contexto
- **Cloud provider:** [AWS / Azure / GCP]
- **Ferramenta IaC:** [Terraform / Pulumi / CDK]
- **Recursos necessários:** [ex: VPC, ECS Cluster, RDS, S3, ALB]

## Tarefa
1. Gere o código IaC para provisionar o ambiente descrito.
2. Organize em módulos reutilizáveis com separação por ambiente.
3. Aplique boas práticas de segurança (IAM mínimo, encryption at rest/transit).
4. Inclua outputs relevantes e documentação dos recursos criados.
